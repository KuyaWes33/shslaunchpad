Place the school logo image in this folder with the exact filename:

    mgc-logo.png

This is the SAME file used by the JHS Launchpad. Just copy the
mgc-logo.png from your JHS Launchpad's "assets" folder into this
"assets" folder, keeping the filename identical.

All pages (index, olive, zion, grading-sheets, login) reference it at:
    assets/mgc-logo.png
— both as the navbar/brand logo and as the browser tab favicon.

A square PNG (roughly 128x128 or larger, transparent background) works best.
